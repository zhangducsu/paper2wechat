# 文章自检 Checklist

## 质检维度 1：类型一（叙事转化风）专属检查
- [ ] **标题质检**：是否包含顶刊名、核心分子，且有一句极具煽动性的价值主张（如“重塑组织溯源”）？
- [ ] **痛点共鸣**：第一段是否把临床医生日常最头疼的痛点（如背景干扰大、灵敏度低）讲透了？
- [ ] **术语白话化**：是否把复杂的生信算法（如深度神经网络降维）用日常比喻或者直接效果做了解释？
- [ ] **设问链有效性**（克制学术风）：开篇是否用了2-3个递进式设问制造知识缺口？设问之间是否有逻辑递进关系（而非简单堆砌）？
- [ ] **对比框架一致性**：全文是否确定了一个核心对比轴？每个子章节是否都回扣了该对比？对比数据是否在多个维度形成了相互印证的证据链？
- [ ] **结论锚点可扫读性**：加粗的结论句是否形成了一条可独立阅读的逻辑链？只扫读加粗句是否能理解全文核心论证？
- [ ] **品牌首尾锚定**：文首（标题区域）和文末（商业段之前）是否都有品牌/公司名露出，形成双重曝光？

## 质检维度 2：类型二（标准八股风）专属检查
- [ ] **档案精准度**：期刊名、IF值、Q区分类是否准确？是否在档案库显眼标出了“技术平台”？
- [ ] **学术规范**：所有的基因/蛋白名称格式是否规范？首次出现的缩写是否都有准确的中文全称？
- [ ] **模块完整度**：是否严格包含了 `[ 小结 ]`、`[ 研究方法 ]`、`[ 结果图形 ]`、`[ 结论和启示 ]` 模块？结果文本是否与 Figure 序号精准对应？

## 质检维度 3：商业闭环检查（类型通用）
- [ ] **引流矩阵**：文章中前部是否插入了同技术方向的“相关阅读”超链接？
- [ ] **顺滑过渡**：结尾部分的“一站式方案/公司业务介绍”是否与前文解读的核心技术完美映射，没有生硬断层？
- [ ] **转化组件**：联系电话、邮箱、企业微信扫码引导、官网等基础设施是否展示完整无缺失？

## 质检维度 4：中文表达与署名规范（类型通用）
- [ ] **中文表达自然度**：解读文本是否符合中文用词、语序和表述习惯，避免英文直译腔？
- [ ] **术语解释本地化**：专业术语是否准确，同时用自然中文解释其意义，而不是逐词翻译英文句子？
- [ ] **英文缩写全称**：专业英文缩写首次出现时是否用括号给出英文全称？有成熟中文译名时是否采用“中文译名（英文全称，英文缩写）”？
- [ ] **作者姓名核验**：疑似中文拼音作者名若被翻译为中文，是否通过网络检索核验？无法核验的作者是否保留英文署名？
